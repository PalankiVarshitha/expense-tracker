# 💰 Expense Tracker

A full-stack expense tracking application built using the MERN stack to manage, analyze, and visualize daily financial transactions.

## 🚀 Features
- Add, update, and delete income and expenses  
- Categorize transactions for better tracking  
- Visual analytics using charts  
- User authentication (login/register)  
- Responsive UI for all devices  

## 🛠 Tech Stack
- MongoDB  
- Express.js  
- React.js  
- Node.js  

## 📂 Project Structure
- client/ → frontend (React application)  
- server.js → backend server  
- routes/ → API routes  
- models/ → database schemas  
- controllers/ → application logic  

## ▶️ Run Locally

Install backend dependencies:
```
npm install
```

Install frontend dependencies:
```
cd client
npm install
```

Run the application:
```
npm run dev
```

## 🎯 Purpose
To build a scalable full-stack application that helps users track expenses, understand spending patterns, and manage finances effectively.
